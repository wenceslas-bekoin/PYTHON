# Bibliothèque flask

## But du projet
Ce projet permet l'utilisation de Python et de Flask.
Le but est d'apprendre les fonctionnalités de Python 
ainsi que celui de Flask.

## Contenu
- `dossier templates`
Pour les fichiers HTML, CSS et JavaScript.
- `app.py`
Contient le code principal de l'application.
- `db.py`
La base de données.
- `scanner.py`
Pour scanner un port sur une addresse ip.

## Utilisation

### Comment lancer l'application
```bash
python3 app.py
```

### Etape suivante
http://127.0.0.1:5000/
